# SoberMotion: Leveraging Probation Force to Reduce the Risk of DUI Recidivism

This is the web interface for visualizing alcohol use behavior.

## Built with

* javascript
* css
* [Chartjs](http://www.chartjs.org)
* [DataTables](https://datatables.net)
